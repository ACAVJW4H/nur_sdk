Linux libraries for various boards.

Files:

libNurApiArmLinux.a			ARM Linux static link library (built in Lubuntu 14.04)
libNurApiArmLinux.so		ARM Linux shared object (built in Lubuntu 14.04)
libNurApiBeagleBone.a		BeagleBone static link library.
libNurApiBeagleBone.so		BeagleBone shared object.
libNurApiRasPi.a			Raspberry Pi static link library.
libNurApiRasPi.so			Raspberry Pi shared object.
